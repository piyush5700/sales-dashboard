# 📱 Phone Sales Dashboard – Power BI Project

This Power BI dashboard provides a complete analysis of mobile phone sales performance using interactive visualizations and business insights. The project helps track sales trends, customer behavior, product performance, and regional sales distribution in an easy-to-understand format.

🚀 Key Features

* 📈 Sales trend analysis using line charts
* 💰 Revenue and profit tracking
* 📊 Product-wise and brand-wise performance analysis
* 🌍 Regional sales insights with map visualization
* 🎯 Sales funnel and customer conversion analysis
* 🥧 Market share representation using pie charts
* 📌 KPI cards for total sales, quantity sold, profit, and customers
* 🔍 Interactive filters and slicers for dynamic reporting

 🛠 Tools & Technologies

* Microsoft Power BI
* Data Cleaning & Transformation using Power Query
* Data Modeling using DAX
* Interactive Dashboard Design & Visualization

 📊 Dashboard Insights

This dashboard helps businesses:

* Identify top-performing mobile brands and products
* Analyze customer purchasing trends
* Monitor sales growth over time
* Compare regional performance
* Make data-driven business decisions

🎯 Project Objective

The main objective of this project is to transform raw sales data into meaningful business insights through data visualization and analytics. The dashboard is designed for decision-makers to quickly understand sales performance and improve strategy.

📂 Project Highlights

* Beginner-to-Intermediate Power BI Project
* Real-world sales analysis use case
* Clean and interactive dashboard design
* Business-focused KPIs and visual storytelling
